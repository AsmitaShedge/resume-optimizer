# Resume Optimizer

## Overview

Resume Optimizer is a Python-based desktop application that analyzes resumes, compares them with Job Descriptions (JD), calculates ATS compatibility, identifies missing skills, generates optimized resumes, and stores records in a MySQL database.

## Features

* Upload Resume (PDF/DOCX)
* Resume Parsing
* Skill Extraction
* Resume Score Calculation
* ATS Score Analysis
* Job Description Matching
* Missing Skill Detection
* Resume Improvement Suggestions
* Generate Optimized Resume
* Download PDF Report
* Export Records to Excel
* Resume Analytics Chart
* MySQL Database Integration
